# Fluxos Principais

## Fluxo: Agendamento de Serviço

1. Operador busca cliente ou cria novo.
2. Define data/horário, escopo e endereço.
3. Sistema valida disponibilidade e sugere equipe.
4. Operador confirma; ordem criada com status "Agendado".
5. Notificações para cliente e equipe.

## Fluxo: Execução e Conclusão

1. Equipe inicia ordem ("Em Execução").
2. Registra ocorrências e materiais utilizados.
3. Conclui ordem; sistema calcula total.
4. Gera fatura e envia ao cliente.

## Fluxo: Reagendamento/Cancelamento

1. Operador solicita alteração.
2. Sistema revalida disponibilidade.
3. Atualiza registro mantendo histórico.
